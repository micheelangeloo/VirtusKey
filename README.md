# VirtusKey Michelangelo Aranguren Briceño || Francisco Manuel Molina Aranda
Proyecto de gestor de contraseñas

VirtusKey será un gestor de contraseñas en el que el usuario tiene una única llave maestra. Solo con esa llave, el sistema desencripta las contraseñas guardadas para que pueda leerlas. Sin la llave, la información es inaccesible.

Buscamos en este proyecto implementar una seguridad real con los datos de un usuario al encriptarlas y ordenarlas en memoria en una base de datos que guarda la información de forma permanente. Además buscamos que sea un proceso sencillo recibir una contraseña, encriptarla, guardarla, recuperarla y mostrarla solo si se pide correctamente esta.

De cara a la siguiente etapa, nos centraremos en desarrollar una interfaz gráfica visualmente atractiva y la lógica del programa.
